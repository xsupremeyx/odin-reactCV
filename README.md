# 📄 CV Builder (React)

A simple and interactive CV (résumé) builder built with React. Users can input their personal, educational, and professional details, preview the formatted CV, and edit it anytime.

---

## 🚀 Features

*  Add General Information (Name, Email, Phone)
*  Add Education Details
*  Add Work Experience
*  Section-wise submit & edit functionality
*  Final validation before generating CV
*  Live CV preview with clean layout
*  Styled using CSS Grid + Flexbox
*  Edit CV after preview

---

## 🧠 Concepts Used

* React Functional Components
* useState Hook
* Controlled Inputs
* Lifting State Up
* Conditional Rendering
* Props & State Management
* CSS Grid & Flexbox

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cv-builder.git
cd cv-builder
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:5173
```

---

## 🧪 How It Works

1. Fill out each section (General, Education, Experience)
2. Click **Submit** inside each section
3. Once all sections are submitted, the **Generate CV** button is enabled
4. Click **Generate CV** to preview your CV
5. Click **Edit CV** to go back and make changes
